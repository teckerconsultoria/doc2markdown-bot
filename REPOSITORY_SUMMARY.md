## 🚀 Como Usar

### Opção 1: Fazer Upload dos Arquivos

```bash
# 1. Crie repositório vazio no GitHub
#    github.com → New repository → doc2markdown-bot

# 2. Clone localmente
git clone https://github.com/seu-usuario/doc2markdown-bot.git
cd doc2markdown-bot

# 3. Copie todos os arquivos de doc2markdown-bot-github/
cp -r /home/claude/doc2markdown-bot-github/* .

# 4. Commit e push
git add .
git commit -m "chore: initial commit with bot and docs"
git push -u origin main
```
